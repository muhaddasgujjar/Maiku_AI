FAQ

Common questions

Is my interview audio stored or sent to any server?
Your audio goes directly to the Groq API (or whichever provider you configure) using your own API key. Maiku AI has no backend server — there is no middleman. Your data travels from your PC to Groq and back, that's it. Groq's free tier does not store audio.
Which AI providers are supported?
Groq (free tier — recommended), OpenAI GPT-4o, Anthropic Claude, and OpenRouter (which gives access to hundreds of models). You switch providers in the Settings tab without restarting the app.
Does it work on Mac or Linux?
The current version is Windows 10/11 only. Windows is required for the WASAPI audio loopback capture (capturing the interviewer's audio from your speakers) and the WDA_EXCLUDEFROMCAPTURE screen-privacy feature. Mac support is planned for a future release.
What's the difference between Free and Pro?
Free is the open-source version on GitHub — you need to be comfortable running Python scripts to set it up. Pro is a polished Windows installer (.exe) with a guided setup wizard, auto-updates, and email support. The AI features and overlay are identical — you're paying for convenience and support, not capability.
Do I need a Groq API key? How much does it cost?
Groq's free tier gives 7,200 seconds of Whisper transcription per day and generous LLM tokens — more than enough for multiple hour-long interviews daily. Creating an account at groq.com is free and takes 2 minutes. No credit card required.
Can the interviewer detect I'm using Maiku AI?
No. The overlay uses Windows' SetWindowDisplayAffinity API to be excluded from all screen capture at the OS level — not by making it transparent or hiding it manually. When you share your screen, the Maiku AI window is genuinely not captured. The interviewer sees only what you'd normally share.
